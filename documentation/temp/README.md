# Temporary Documentation

This folder is for temporary documentation files that are not permanent project artifacts.

## What belongs here

- Debugging notes and investigation findings
- Draft documentation during development
- Temporary analysis files
- Short-lived planning documents

## What does NOT belong here

- Permanent documentation (use `DOC_*.md` prefix in `documentation/` root)
- Feature implementation docs (use `documentation/jobs/`)
- Reference material (use subdirectories in `documentation/`)

## Cleanup

Files in this folder can be safely deleted when no longer needed.
